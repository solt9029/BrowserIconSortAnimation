# 環境構築

- node: 6.11.1

- npm: 3.10.10

```
cd BrowserIconSortAnimation
npm install
```

ファイルの変更を監視する

```
npm run watch
```

http://localhost:8080 から確認可能にする（ローカルサーバ）

```
npm run serve
```

# 使用方法

ウィンドウの幅を変更した時のアイコン整列手法の違いを見れる

デフォルトではウィンドウ幅を一気に広げたときでも1段階ずつアニメーションする

キーを押すことで、一気に広げたときに一気にアニメーション出来るように変更できる
